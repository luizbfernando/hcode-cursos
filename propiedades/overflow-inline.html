Propriedade overflow-block

Disponível apenas no firefox

Esta propriedade CSS irá especificar o que deve ser exibido quando o conteúdo estoura os limites de um elemento de linha onde foi definida e pode ser usada na condição de uma media query

Esta propriedade irá direcionar para as propriedades overflow-y ou overflow-x dependendo do modo de escrita do documento

Valores

Valores possíveis:Visible, hidden, scroll, auto, inherit, initial, unset

Valor Padrão:auto

Valor Herdado:não
